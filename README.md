## Test Frameworks

### Java / Kotlin
- [JUnit 5](https://github.com/junit-team/junit5) - The next generation of JUnit.
- [TestNG](https://github.com/testng-team/testng) - Testing framework inspired by JUnit and NUnit with new features.
- [Kotest](https://github.com/kotest/kotest) - Flexible and elegant Kotlin multiplatform test framework.

### JavaScript / TypeScript
- [Jest](https://github.com/facebook/jest) - Delightful JavaScript testing with a focus on simplicity.
- [Vitest](https://github.com/vitest-dev/vitest) - Blazing-fast Vite-native unit testing framework.
- [Mocha](https://github.com/mochajs/mocha) - Simple, flexible, fun JavaScript test framework for Node.js.

### Python
- [pytest](https://github.com/pytest-dev/pytest) - The most popular Python testing framework.
- [ward](https://github.com/darrenburns/ward) - A modern test framework for Python with a focus on productivity.

### Go
- [testify](https://github.com/stretchr/testify) - A toolkit with common assertions and mocks for Go.
- [ginkgo](https://github.com/onsi/ginkgo) - BDD testing framework for Go.

### .NET
- [xUnit](https://github.com/xunit/xunit) - Free, open-source, community-focused unit testing tool for .NET.
- [NUnit](https://github.com/nunit/nunit) - A unit-testing framework for all .NET languages.

## API Testing

- [REST Assured](https://github.com/rest-assured/rest-assured) - Java DSL for easy testing of REST services.
- [Supertest](https://github.com/ladjs/supertest) - Super-agent driven library for testing HTTP servers in Node.js.
- [Hoppscotch](https://github.com/hoppscotch/hoppscotch) - Open-source API development ecosystem.
- [Bruno](https://github.com/usebruno/bruno) - Offline-first, Git-friendly API client.
- [httpie](https://github.com/httpie/cli) - Modern command-line HTTP client for API testing.
- [k6](https://github.com/grafana/k6) - Modern load testing tool using JavaScript (also in Performance).
- [Karate](https://github.com/karatelabs/karate) - API test-automation, mocks, performance-testing, and UI automation in one DSL.

---

## UI & Browser Automation

- [Playwright](https://github.com/microsoft/playwright) - Reliable end-to-end testing for modern web apps across all browsers.
- [Cypress](https://github.com/cypress-io/cypress) - Fast, easy and reliable testing for anything that runs in a browser.
- [Selenium](https://github.com/SeleniumHQ/selenium) - Browser automation framework and ecosystem.
- [WebdriverIO](https://github.com/webdriverio/webdriverio) - Next-gen browser and mobile automation test framework for Node.js.
- [Puppeteer](https://github.com/puppeteer/puppeteer) - Node.js library for controlling Chrome via the DevTools Protocol.

---

## Mobile Testing

- [Appium](https://github.com/appium/appium) - Automation framework for native, hybrid, and mobile web apps.
- [Detox](https://github.com/wix/Detox) - Gray-box end-to-end testing for React Native apps.
- [XCUITest](https://developer.apple.com/documentation/xctest) - Apple's native UI testing framework for iOS.
- [Espresso](https://developer.android.com/training/testing/espresso) - Android UI test framework from Google.
- [maestro](https://github.com/mobile-dev-inc/maestro) - The simplest and most effective mobile UI testing framework.

## Performance & Load Testing

- [k6](https://github.com/grafana/k6) - Developer-centric load testing tool with JavaScript scripting.
- [Gatling](https://github.com/gatling/gatling) - Load testing tool with Scala DSL and detailed HTML reports.
- [Locust](https://github.com/locustio/locust) - Scalable load testing in Python.
- [Apache JMeter](https://github.com/apache/jmeter) - Java-based load and performance measurement tool.
- [Artillery](https://github.com/artilleryio/artillery) - Cloud-scale performance testing for modern apps.

## Contract Testing

- [Pact](https://github.com/pact-foundation/pact-js) - Consumer-driven contract testing framework.
- [Spring Cloud Contract](https://github.com/spring-attic/spring-cloud-contract) - Contract verification for Spring applications.

## Security Testing

- [OWASP ZAP](https://github.com/zaproxy/zaproxy) - World's most widely used web app scanner.
- [Burp Suite Community](https://portswigger.net/burp/communitydownload) - Leading toolkit for web security testing.
- [Trivy](https://github.com/aquasecurity/trivy) - Vulnerability scanner for containers, filesystems, and repositories.
- [semgrep](https://github.com/semgrep/semgrep) - Lightweight static analysis for many languages.

## Test Data & Mocking

- [Faker.js](https://github.com/faker-js/faker) - Generate massive amounts of realistic fake data in Node.js.
- [Faker (Python)](https://github.com/joke2k/faker) - Python package that generates fake data.
- [WireMock](https://github.com/wiremock/wiremock) - Flexible API mocking tool for testing.
- [Mockoon](https://github.com/mockoon/mockoon) - Easy and fast mock servers, desktop and CLI.
- [json-server](https://github.com/typicode/json-server) - Full fake REST API with zero coding.
- [Testcontainers](https://github.com/testcontainers/testcontainers-java) - Lightweight, throwaway instances of common databases and services via Docker.

## Reporting & Observability

- [Grafana](https://github.com/grafana/grafana) - Observability platform for metrics, logs, and traces.
- [Mochawesome](https://github.com/adamgruber/mochawesome) - Custom reporter for the Mocha JavaScript test framework.

## CI/CD & Infrastructure

- [GitHub Actions](https://github.com/features/actions) - Automate workflows directly in GitHub.
- [Jenkins](https://github.com/jenkinsci/jenkins) - Leading open-source automation server.
- [GitLab CI](https://docs.gitlab.com/ee/ci/) - Built-in continuous integration in GitLab.
- [Docker](https://github.com/docker/compose) - Container platform for building reproducible test environments.

## AI-Assisted Testing

- [CodiumAI](https://github.com/Codium-ai/pr-agent) - AI-powered code review and test generation.
- [Diffblue Cover](https://www.diffblue.com/) - Autonomous Java unit test writing using AI.
- [Applitools](https://applitools.com/) - AI-powered visual testing and monitoring.

## AI Agents for Testing

- [Browser Use](https://github.com/browser-use/browser-use) - Open-source agent that controls browsers using natural language instructions.
- [Stagehand](https://github.com/browserbase/stagehand) - AI-native browser automation framework built on Playwright by Browserbase.
- [ZeroStep](https://github.com/zerostep-ai/zerostep) - Natural language AI actions that run directly inside Playwright tests.
- [Shortest](https://github.com/anti-work/shortest) - AI-first end-to-end testing via natural language test definitions.
- [AgentQL](https://github.com/tinyfish-io/agentql) - Query language and agent for natural language web automation and scraping.
